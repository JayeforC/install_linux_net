## Configure file for FDU net login and logout for Ubuntu ##

### 1. Clone this repository ###

### 2. Follow the commands ###
```
cd install_linux_net

bash install_linux_net.sh

```
### 3. Usage
```
# login
onnet 

# logout
offnet
